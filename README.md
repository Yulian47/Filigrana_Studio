# Filigrana Studio - Landing Page

Сучасна, адаптивна лендінг сторінка для дизайнерської студії Filigrana Studio.

## Можливості

- 📱 Повністю адаптивний дизайн (мобільне-перший підхід)
- 🎨 Сучасний дизайн з анімаціями
- ⚡ Виразні переходи та ефекти наведення
- 🔍 SEO-оптимізований HTML
- 📦 Структурований код
- 🎯 Гладкий скролінг та навігація

## Структура проекту

```
filigrana-studio/
├── index.html           # Основний HTML файл
├── css/
│   └── styles.css       # Скомпільовані стилі
├── scss/
│   └── styles.scss      # SCSS вихідний файл
├── js/
│   └── script.js        # JavaScript функціональність
├── assets/              # Папка для зображень та медіа
└── README.md            # Цей файл
```

## Технологічний стек

- **HTML5** - Семантична розмітка

- **SCSS** - Препроцесор CSS для кращої організації стилів
- **JavaScript (ES6+)** - Інтерактивність та функціональність

## Введення в роботу

### 1. Клонування або завантаження проекту

```bash
cd filigrana-studio
```

### 2. Компіляція SCSS (якщо потрібно)

Якщо ви хочете редагувати SCSS напряму, вам знадобиться компілювати його в CSS:

```bash
# Установка sass (якщо не встановлено)
npm install -g sass

# Компіляція SCSS в CSS
sass scss/styles.scss css/styles.css

# Спостереження за змінами SCSS
sass --watch scss:css
```

### 3. Запуск на локальному сервері

```bash
# Використовуючи Python 3
python3 -m http.server 8000

# Або використовуючи Node.js (якщо встановлено)
npx http-server

# Або використовуючи Live Server в VS Code
# Встановіть розширення Live Server і натисніть на іконку
```

Потім відкрийте у браузері: `http://localhost:8000`

## Налаштування контенту

### Зміна логотипу

Відредагуйте текст логотипу в файлі `index.html`:

```html
<a href="#" class="logo">Filigrana Studio</a>
```

### Додавання зображень портфеліо

Замініть шляхи зображень в секції Portfolio:

```html
<img src="assets/your-image.jpg" alt="Project Name" />
```

### Редагування контактної інформації

Модифікуйте форму контакту в секції Contact або додайте реальний обробник форми.

## Кастомізація дизайну

### Колірна схема

Відредагуйте змінні в `scss/styles.scss`:

```scss
$primary-color: #6c5ce7; // Основний фіолетовий колір
$secondary-color: #a29bfe; // Вторинний колір
$text-color: #2d3436; // Колір тексту
$light-bg: #f5f5f5; // Світле фоне
```

Потім скомпілюйте SCSS.

### Шрифти

Змініть сімейство шрифтів в `css/styles.css`:

```css
body {
  font-family: "Segoe UI", Tahoma, Geneva, Verdana, sans-serif;
}
```

Додайте нові `<section>` елементи в HTML, стилізуйте їх в CSS/SCSS, і додайте функціональність в JS.

Замініть обробник форми контакту на справжній API запит:

```javascript

  },
});
```
